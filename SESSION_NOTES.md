**Auto-hide for the defer row.**
Removed the mobile rule that kept the defer row permanently visible under every item — was making the screen look busy. Now the row is hidden by default on every device and revealed only when you focus on an item (hover on desktop, tap on mobile). After it appears, a 4-second timer starts; if you don't pick a destination, the row collapses again. Tapping a defer/delete button collapses immediately because the action triggers a re-render. Also added a guard so only one row is expanded at a time — tap a new row and the previously open one closes. Auto-hide window is `EXPAND_AUTO_HIDE_MS = 4000` if we want to tune it.

**Tightened spacing again + enabled spellcheck.**

**Reminders Today → app sync, via Shortcut.**
